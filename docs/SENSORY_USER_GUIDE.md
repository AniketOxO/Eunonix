## üéÆ Using the Controls

### Selecting a Profile
1. **Click any profile card** in the grid
2. Your phone will **vibrate** (if supported)
3. **Audio starts playing** automatically
4. **3D environment** begins rendering
5. **Controls appear** below for customization

### Adjusting Brightness
- Slide the **brightness control** (0-100%)
- Higher = brighter ambient light effect
- Lower = dimmer, more subdued
- **Recommended**: 40-60% for comfort

### Changing Colors
- Click any of the **5 color circles**
- Selected color has a **thick border**
- Changes the ambient light hue
- **Tip**: Match your room lighting

### Volume Control
- Use the **volume slider** (0-100%)
- Adjusts sound in **real-time**
- Click **Play/Stop** to toggle audio
- **Recommended**: 30-50% for background

### Sound Types
1. **Nature Sounds** - Pink noise, like wind/rain
2. **Ambient Music** - Calming sine waves
3. **Binaural Beats** - Focus-enhancing dual frequencies
4. **White Noise** - Pure random noise, masks distractions
5. **Calming Music** - 432Hz healing frequency

### Haptic Feedback (Mobile/Tablet)
- Adjust **intensity slider** (0-100%)
- Try patterns with **test buttons**:
  - **Calm**: Gentle, slow pulses
  - **Energize**: Quick, rapid vibrations
  - **Ground**: Deep, strong vibrations
  - **Focus**: Rhythmic, steady pattern

### 3D Environments
Click any environment to **switch instantly**:
- **Ocean** - Animated waves, blue gradient
- **Cosmos** - Stars, nebula, space
- **Forest** - Trees, fireflies, nature
- **Desert** - Sand dunes, sunset
- **Aurora** - Northern lights, colors
- **Abstract** - Particles, color cycling

The **preview canvas** updates immediately!

## üîß Troubleshooting

### Audio Not Playing?
- Click the **Play** button next to "Soundscape"
- Check your **device volume** is not muted
- Try a different **sound type**
- Refresh the page

### Haptics Not Working?
- Check if your device supports vibration
- iOS requires **user interaction** first (tap test button)
- Not supported on desktop browsers
- Try clicking a **test button** again

### 3D Environment Frozen?
- Switch to a **different environment**
- Refresh the page
- Check if another tab is using too much CPU

### Immersive Mode Won't Exit?
- Press **ESC** key
- Click the **√ó button** (top right)
- Press **F11** to exit fullscreen manually

### Profile Not Activating?
- Click the profile card again
- Refresh the page
- Check browser console for errors
